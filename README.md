**Sales-Data-Analysis-Dashboard**

**Overview**

This project demonstrates a Sales Data Analysis Dashboard using Excel and Power BI. It focuses on cleaning, visualizing, and analyzing sales datasets to provide actionable insights for businesses.

## Features

* Data cleaning and preprocessing in Excel
* Interactive dashboards in Power BI
* Visualization of key metrics: Sales, Profit, Region-wise Performance
* Exportable insights for reporting and decision-making

## Dataset

The sales dataset includes columns like:

* Date of Sale
* Product Category
* Sales Amount
* Profit
* Region

You can find the dataset in the file: Sales_Data.csv

## Files in this Repo

* Sales_Data.csv — Raw dataset
* Sales_Data_Analysis_Dashboard.pbix — Power BI dashboard file
* Sales_Data_Analysis_Report.pdf — Project report
* Sales-Data-Analysis-Dashboard.png — Screenshot of the dashboard

## How to Use

1. Open the Power BI file (.pbix) in Power BI Desktop.
2. Load the dataset if not already loaded.
3. Interact with the dashboard to explore sales insights.
4. Export visuals or reports as needed.
