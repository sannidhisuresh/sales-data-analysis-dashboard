Sales Data Insights

Power BI • Excel • Data Cleaning • Data Visualization

This project presents a comprehensive Sales Data Analysis Dashboard built using Power BI.
It helps businesses monitor performance metrics, identify top-selling products, analyse revenue trends, and understand regional performance.

Project Overview

The dashboard provides insights on:

1. Total Revenue
2. Total Profit
3. Monthly Revenue Trend
4. Top-Selling Products
5. Category-wise Sales
6. Region-wise Revenue

Tools & Technologies Used

1. Power BI Desktop (.pbix)
2. Excel / CSV (Dataset)
3. Power Query (Data Cleaning)
4. DAX (Basic Calculations)
5. Data Visualization Techniques

Key Insights

1. The business generated 14M Total Revenue.
2. Total profit recorded is 10M.
3. Accessories contribute 56% of total category sales.
4. Regional revenue indicates major contributions from America, Europe, and Africa.
5. Top-selling products can be identified from the product-wise analysis chart.
